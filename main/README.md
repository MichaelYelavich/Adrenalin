## -ADRENALIN-

This is a plugin for the [SteamOS Plugin Loader](https://github.com/SteamDeckHomebrew/PluginLoader).

## ADRENALIN is a pure performance and optimization tool that aims to fade into the background and do everything for you. 

Adrenalin should automatically act to improve efficiency through improving memory addressing and delegate the most efficient CCX configuration for the current active process. Additionally, the more advanced user can modify the VCore offset, useful for undervolting which is often a huge perf and efficiency win. 

##With those features used in tandem, your deck gets Adrenalin.

## Features

- DDR Gear Down Mode for Power Saving
- DDR Manipulation (pending hardware exposure)
- Transparent Huge Pages
- Load adaptive clustered multithreading (WIP) (pending cppc not in old deck kernel)
- Independent SoC voltage offset (WIP) (pending governor)
